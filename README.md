# Introduction
Vim custom tab line with tab number.  
![vim-tabline](_assets/tabline.png)  
# Installation
Vundle `Plugin 'xuxinx/vim-tabline'`
# Settings
```vim
let g:tabline_show_wins_count = 1 " show windows count, default is disabled
```
